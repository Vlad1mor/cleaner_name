# Russian Name Cleaner

Очистка имён от лишних символов (пробелов, пунктуации, латиницы, цифр) с загрузкой в PostgreSQL.

## Возможности

- Читает файлы **CSV**, **Excel (.xlsx, .xls)**, **JSON**.
- Оставляет в строковых полях **только русские буквы** (включая Ёё).
- Удаляет записи, которые после очистки стали пустыми.
- Загружает результат в таблицу PostgreSQL (создаёт или перезаписывает).

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/russian-name-cleaner.git
   cd russian-name-cleaner
   ```

2. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux/Mac
   venv\Scripts\activate      # Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Настройка PostgreSQL

- Убедитесь, что PostgreSQL запущен и создана база данных.
- Подготовьте строку подключения:  
  `postgresql://user:password@localhost:5432/database_name`

### (Опционально) Переменные окружения

Создайте файл `.env` в корне проекта:

```
DB_USER=postgres
DB_PASSWORD=your_password
DB_HOST=localhost
DB_PORT=5432
DB_NAME=your_database
```

Скрипт загрузит их автоматически (используется `python-dotenv`).

## Использование

1. Положите исходный файл (CSV, Excel или JSON) в папку `data`.
2. При необходимости отредактируйте настройки в `src/main.py`:
   - `DB_URL` – строка подключения к PostgreSQL.
   - `TABLE_NAME` – имя таблицы, куда сохранять.
   - Расширение файла (по умолчанию ищет первый `.csv` – можно изменить).
3. Запустите скрипт из корня проекта:
   ```bash
   python src/main.py
   ```

Пример вывода:
```

## Как это работает

1. Модуль `cleaner.keep_only_russian_letters` удаляет всё, кроме символов `а-яА-ЯёЁ`.
   - `"Воро бьев"` → `"Воробьев"`
   - `"Дан,иил"` → `"Даниил"`
   - `"Степан.ович"` → `"Степанович"`
2. Все строковые колонки DataFrame проходят через эту функцию.
3. Строки, где после очистки не осталось ни одного символа, удаляются.
4. Очищенный DataFrame сохраняется в PostgreSQL с помощью `pandas.to_sql`.

## Пример входных данных

**Файл `data/people.xlsx`**

| full_name          | city         |
|--------------------|--------------|
| Воро бьев          | Москва       |
| John Смирнов       | СПб          |
| Дан,иил            | 12345        |
| Степан.ович        | Казань       |

**После очистки и загрузки в таблицу `cleaned_names`**:

| full_name    | city    |
|--------------|---------|
| Воробьев     | Москва  |
| Смирнов      | СПб     |
| Даниил       |         |
| Степанович   | Казань  |

> Строка `John Смирнов` превратилась в `Смирнов` (латиница удалена), запись с `12345` стала пустой по колонке `full_name` и была удалена.

## Расширение

- Чтобы сохранять дефисы (`Анна-Мария`) – измените регулярное выражение в `cleaner.py` на `r'[^а-яА-ЯёЁ\-]'`.
- Для обработки всех файлов в `data/` – добавьте цикл в `main.py`.
- Чтобы загружать данные с добавлением (`append` вместо `replace`) – измените параметр `if_exists`.
